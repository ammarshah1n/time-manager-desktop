# Timed

[![macOS CI](https://github.com/ammarshah1n/time-manager-desktop/actions/workflows/macos-ci.yml/badge.svg)](https://github.com/ammarshah1n/time-manager-desktop/actions/workflows/macos-ci.yml)
[![Latest Release](https://img.shields.io/github/v/release/ammarshah1n/time-manager-desktop)](https://github.com/ammarshah1n/time-manager-desktop/releases/latest)

Timed is an **executive operating system** for a single C-suite user (Yasser Shahin). It is not a productivity app. It builds a compounding cognitive model of how one executive thinks and operates, and returns cognitive bandwidth through a nightly reflection engine and a morning intelligence briefing.

> **Mission realignment 2026-04-02:** the original Year-11 study planner framing has been retired. Timed is now the macOS client for the intelligence engine described in `MASTER-PLAN.md`.

## What it is

- macOS 15 SwiftUI desktop app, single-user, native-only (no web, no iOS in v1).
- Ingests Outlook email + calendar via Microsoft Graph (MSAL OAuth).
- Nightly Opus 4.6 reflection engine: raw observations → first-order patterns → second-order synthesis → semantic model → procedural rules (Stanford Generative Agents architecture, Park et al. 2023).
- Morning briefing: 7-section CIA PDB format, generated by Opus at max effort.
- 5-tier memory: raw observations / daily summaries / behavioural signatures / personality traits + ACB-FULL and ACB-LIGHT active-context buffers.
- Cognitive layer only — observes, reflects, recommends. Never acts on the world unilaterally.

## Design principles

1. **No cost cap on intelligence.** Opus 4.6 at max effort for every durable-memory generation call. Budget $800/week, actual ~$63/week.
2. **Intelligence compounds over time.** Month 6 >> Month 1. This is the moat.
3. **Nightly engine is the heart**, not a nice-to-have feature.
4. **Morning session delivers intelligence**, not a task list.
5. **User is a C-suite executive.** Impressed only by a system that understands him.

## Architecture (current state)

- **UI:** SwiftUI + TCA (`@Dependency` throughout; `@Reducer` in `IntroFeature`)
- **Auth:** `AuthService.swift` — Supabase Auth + Microsoft OAuth, 30 call sites (UI bridge is next priority)
- **Email:** `EmailSyncService` + `GraphClient` — delta sync + webhook renewal
- **Backend:** Supabase project `fpmjuufefhtlwbfinxlx`, 29 Edge Functions ACTIVE, 48 SQL migrations
- **Embeddings:** Voyage `voyage-3` (Tier 0, 1024-dim) + OpenAI `text-embedding-3-large` (Tier 1–3, 3072-dim)
- **Orchestration:** pgflow over pgmq, 4-cron schedule (full / refresh / briefing / weekly pruning)

## Trust boundary

- AI routed only via Supabase Edge Functions — no direct client-to-model calls.
- Secrets (Anthropic, Voyage, OpenAI, Graph) live in Supabase secrets + 1Password.
- Timed observes and recommends; the executive always approves world-changing actions.
