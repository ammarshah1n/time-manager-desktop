# 00 — Project Overview

## Product Vision

Timed is the most intelligent executive operating system ever built.

It is not a productivity app. It is not a task manager. It is a system that
builds a deep, compounding cognitive model of how a specific C-suite executive
thinks, decides, avoids, prioritises, communicates, and operates — and uses
that model to give them their cognitive bandwidth back, every single day,
permanently.

## Core Thesis

Intelligence compounds over time. Every day generates more data. The reflection
engine synthesises that data into deeper understanding. Every morning, that
understanding makes the plan smarter. Month 6 is incomparably more intelligent
than month 1. This compounding is the moat.

## Product Boundary (non-negotiable)

Timed is a COGNITIVE INTELLIGENCE LAYER.
- It observes, reflects, and recommends
- The user always decides and executes
- No unilateral calendar actions, no sending emails, no acting on the world
- This constraint is the product — it eliminates the failure mode that killed
  every action-oriented AI in this space (Humane, Rabbit, Clara, x.ai)

## Target User

C-suite executive who:
- Lives in Microsoft Outlook (email + calendar)
- Has tried every productivity tool and is not impressed by features
- Has hired coaches, has a PA, still feels cognitively overloaded
- Would be impressed by a system that understands them in a way nothing else
  does, and gets measurably smarter the longer they use it

## Value Architecture (from Research Pack 01)

### Tier 1 — Operational (months 1-2)
Task prioritisation that learns this person. Email triage that improves with
corrections. Calendar scheduling that protects peak cognitive windows. Time
estimation calibrated to actual completion patterns.

### Tier 2 — Intelligence (months 3-6)
Morning briefing opens with named patterns. Avoidance detection with historical
context. Chronotype-calibrated scheduling. Relationship health signals.

### Tier 3 — Compounding Strategic (months 6+)
Cognitive model so specific it surfaces patterns the executive has never
consciously recognised. Year-over-year decision quality comparisons. Queryable
personal intelligence: "Show me the conditions under which I've made my best
decisions in the last 18 months."

## The Conversion Moment
Month 3: the morning session opens with a pattern about the executive's
behaviour that is exactly right — specific, named, actionable, something they
had never put into words. That moment creates a category of one.
