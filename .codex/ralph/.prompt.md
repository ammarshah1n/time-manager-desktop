**Acceptance criteria:**
- A 'Study' button per subject task opens a full-screen modal (not a sheet) with dark glassmorphic overlay

- Each question shows: question text, 4 multiple-choice options (A/B/C/D), a text field for free-form answer, and 'Submit' button
- On submit: show correct answer highlighted green, incorrect answer highlighted red, 1-2 sentence explanation, source note title
- After 5+ questions: show session summary — X/Y correct, confidence delta (+N%), 'End Session' or 'Keep Going' buttons
- On 'End Session': update task.confidence in PlannerStore, trigger rebuildPlan()
- App builds without errors

**Implementation notes:**
Create a new QuizView.swift (or extend the existing quiz UI in ContentView). Key components: (1) QuizSessionModel: @Observable class holding currentQuestion, questionHistory, sessionScore, subject. (2) QuizQuestionView: shows question + 4 options as large tappable cards. (3) Question generation: call CodexBridge with a prompt like 'Generate 5 quiz questions from this content: [context]. Return JSON array of {question, options: [A,B,C,D], correct: A, explanation, source}. (4) Parse JSON response to [QuizQuestion] model.
